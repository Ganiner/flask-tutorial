环境依赖:
```
pip install flask
pip install flask-script
pip install flask-bootstrap
pip install flask-wtf
pip install flask-sqlalchemy
pip install flask-login
```

如何使用：
```
git clone git@github.com:LToddy/blog.git
```
然后解压，打开,在命令行终端输入:
```
python3 manage.py runserver
```
